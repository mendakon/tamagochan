## Requirement
node v12.14.0

## Installation
```
$git clone https://github.com/mendakon/tamagochan.git
$cd tamagochan
$npm install
$npm install -g forever
```

## Usage
```
$node authorization.js
$node init.js
$forever start init.js
```
